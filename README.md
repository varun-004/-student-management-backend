# Student Management System

## 🚀 Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Swagger

## 📌 Features
- CRUD operations
- DTO (Request & Response)
- Validation using @Valid
- Global Exception Handling
- Pagination
- Search + Pagination
- Sorting
- Swagger API documentation

## ▶️ How to Run
1. Clone the repository
2. Configure database in application.properties
3. Run the application
4. Open Swagger:
   http://localhost:8080/swagger-ui/index.html

## 📖 Description
This project is a REST API for managing student data.
It follows layered architecture (Controller → Service → Repository)
and includes best practices like DTO, validation, and exception handling.
